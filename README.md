# Hangdogger
Code for an iPhone app that tracks hangboard workouts and finger injuries. Contains in interval time tab for hangboarding workouts (or anything esle), 
a history tab where you can upload or download data in .csv files or add individual workouts, and injuries tab to enter your injurys, and a progress
tab where you can visualize your workouts and injuries. Presently assumes every workout is either a max hangs workout or a 7:3 repeaters workout.
